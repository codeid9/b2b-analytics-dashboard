# AI-Powered B2B Analytics Dashboard

A modern, highly responsive SaaS B2B Analytics Dashboard designed and engineered to showcase optimized data visualization workflows, design system principles, and context-aware simulated AI capabilities. 

## 🚀 Key Features

- **Dynamic KPIs & Analytics:** High-impact business metrics featuring performance trend vectors and clean interactive data structures.
- **Interactive Data Visualization:** Custom telemetry line graphs, bar charts, and demographic distribution layouts tracking global workspace revenue.
- **Simulated AI Copilot Page:** A deterministic natural language interface demonstrating robust text-processing mock pipelines.
- **Enterprise Registry Management:** Client roster filtering with client-side searching logic and custom context status indicators.
- **Responsive Fluid UI:** Adaptive desktop navigation combined with mobile-first persistent bottom navigation layouts.
- **Design System Consistency:** Component structures built on reusability, consistent token states, and fluid interface mechanics.

## 🛠️ Tech Stack

- **Framework:** Next.js (App Router, TypeScript)
- **Styling Config:** Tailwind CSS v4 Engine
- **UI Components:** shadcn/ui primitives
- **Data Charting:** Recharts Ecosystem
- **Icon Assets:** Lucide React
- **Motion Engine:** Framer Motion

## 📁 Architecture & File Directory

The codebase follows an atomic, flat directory model designed for optimal discovery and strict relative path linking:

```text
src/
 ├── app/                  # Route layouts, application routing hooks
 │    ├── dashboard/       # Landing view with core business modules
 │    ├── analytics/       # Advanced visual charts and demographic views
 │    ├── ai-copilot/      # Simulated intelligence processing views
 │    ├── customers/       # Search-enabled registry records
 │    ├── reports/         # Operational audit document downloads
 │    └── settings/        # Webhook definitions and form controls
 ├── components/           # Extensible structural view elements
 │    ├── ui/              # Decoupled component interface systems
 │    ├── sidebar/         # Responsive viewports and persistent links
 │    ├── charts/          # Multi-axis Recharts components
 │    └── cards/           # Animated KPI blocks
 ├── data/                 # Structural mock telemetry matrices
 └── lib/                  # Shared system configuration variables
```

 ⚡ Getting Started Locally

1. Clone the repository
```Bash
git clone https://github.com/codeid9/b2b-analytics-dashboard.git
cd b2b-analytics-dashboard
```

2. Install dependencies
```Bash
npm install
```

3. Run the local development server
```Bash
npm run dev
```

Open http://localhost:3000 with your browser to see the live output.
